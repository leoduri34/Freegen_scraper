# Freegen_scraper
This is an image generator that uses the "playwright" python library to open the ai image generator "freegen.app", generate an image with the given prompt, and scrapes the image. It's a little rusty around the edges, so I'd love if anybody has advice or can help improve it!

You can use the generate() function by entering a prompt as a string, and entering in the save file name, which defaults to generate.png. Happy generating!
